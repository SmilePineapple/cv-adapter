Homepage layout

Minimal monochrome direction. Pick the structure that feels easiest to navigate.

A

Single-page scroll

Homepage contains About + Services + Featured work + Contact sections. Nav scrolls to sections.

B

Multi-page site

Home is a summary. Dedicated pages: Portfolio, About, Services, Contact. Cleanest for SEO + depth.

C

Hybrid (recommended)

Home is a scrollable summary + strong CTA. Portfolio + Contact are dedicated pages for focus.

Tip: If you’ll have lots of projects, multi-page or hybrid typically feels more “premium”.
