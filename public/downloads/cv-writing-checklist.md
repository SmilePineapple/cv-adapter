# CV Writing Checklist - My CV Buddy

## Before You Start
- [ ] Read the job description thoroughly
- [ ] Highlight key requirements and keywords
- [ ] Gather your employment history and dates
- [ ] List your qualifications and certifications
- [ ] Prepare examples of achievements with numbers

---

## Contact Information
- [ ] Full name (professional format)
- [ ] UK mobile number (07XXX XXXXXX format)
- [ ] Professional email address (firstname.lastname@email.com)
- [ ] Location (City, UK - no full address)
- [ ] LinkedIn profile URL (optional but recommended)
- [ ] Portfolio/website (if relevant)

### ❌ DO NOT Include:
- [ ] Photo (unless acting/modeling)
- [ ] Date of birth or age
- [ ] Marital status or children
- [ ] National Insurance number
- [ ] Full home address

---

## Professional Summary (3-5 sentences)
- [ ] Job title and years of experience
- [ ] Key skills and areas of expertise
- [ ] 1-2 major achievements with numbers
- [ ] What you're looking for (optional)
- [ ] Tailored to the specific job

---

## Work Experience
For each role, check:
- [ ] Job title clearly stated
- [ ] Company name and location
- [ ] Employment dates (Month Year format)
- [ ] 3-6 achievement-focused bullet points
- [ ] Quantified results (%, £, numbers)
- [ ] Action verbs (Led, Managed, Achieved, Implemented)
- [ ] Keywords from job description included
- [ ] Most recent role has most detail (5-6 bullets)
- [ ] Older roles have less detail (2-3 bullets)

### Achievement Formula:
- [ ] Action verb + Task + Result with numbers
- [ ] Example: "Increased sales by 35% through data-driven strategy"

---

## Education
- [ ] Degree classification and subject
- [ ] University name and graduation year
- [ ] A-Levels with grades (if recent graduate)
- [ ] GCSEs summary (if recent graduate)
- [ ] Relevant modules (only if graduate/recent grad)
- [ ] Academic achievements (Dean's List, awards)

### For Experienced Professionals (5+ years):
- [ ] Keep education brief (one line per qualification)
- [ ] Remove A-Levels and GCSEs

---

## Skills Section
- [ ] Technical skills relevant to the role
- [ ] Software and tools you're proficient in
- [ ] Professional/industry-specific skills
- [ ] Languages with proficiency levels
- [ ] Certifications and qualifications
- [ ] Skills match job description keywords
- [ ] No generic skills ("hard worker", "team player")
- [ ] Specific, not vague (e.g., "Excel Advanced" not "computer skills")

---

## Optional Sections (Include if Relevant)
- [ ] Certifications (PMP, ACCA, AWS, CIPD, etc.)
- [ ] Professional memberships
- [ ] Volunteering (if demonstrates relevant skills)
- [ ] Projects (especially for tech/creative roles)
- [ ] Awards and achievements
- [ ] Publications or speaking engagements
- [ ] Interests (only if unique or relevant)

---

## Formatting & Design
- [ ] Professional font (Arial, Calibri, Times New Roman)
- [ ] Font size 10-12pt for body text
- [ ] Font size 14-16pt for headings
- [ ] Consistent formatting throughout
- [ ] 2.5cm margins on all sides
- [ ] Line spacing 1.15 or 1.5
- [ ] Clear section headings
- [ ] Adequate white space
- [ ] No tables or text boxes (ATS can't read)
- [ ] No headers/footers with important info
- [ ] 2 pages maximum (UK standard)

### Color Guidelines:
- [ ] Conservative industries: Black/grey only
- [ ] Professional roles: One accent color acceptable
- [ ] Creative roles: Modern color palette OK
- [ ] Never use bright/neon colors

---

## ATS Optimization
- [ ] Standard section headings ("Work Experience" not "Where I've Worked")
- [ ] Keywords from job description included naturally
- [ ] No graphics, charts, or images
- [ ] Simple bullet points (•)
- [ ] No columns or complex layouts
- [ ] Saved as .docx or PDF (check job posting preference)
- [ ] File named: FirstName_LastName_CV.pdf

---

## Content Quality
- [ ] No typos or grammatical errors
- [ ] No first person ("I", "my", "me")
- [ ] Active voice, not passive
- [ ] Specific, not vague
- [ ] Achievements, not just responsibilities
- [ ] Numbers and percentages included
- [ ] Industry-specific terminology used
- [ ] Tailored to the specific job
- [ ] No unexplained employment gaps
- [ ] Consistent date formatting throughout

---

## Final Checks
- [ ] Read aloud to catch errors
- [ ] Spell check completed
- [ ] Grammar check completed
- [ ] Asked someone else to proofread
- [ ] Checked all dates are accurate
- [ ] Verified all contact information is correct
- [ ] Tested PDF opens correctly
- [ ] File size under 2MB
- [ ] Printed version looks professional
- [ ] Matches job description requirements

---

## Tailoring for This Application
- [ ] Professional summary mentions company/role
- [ ] Keywords from job description in CV
- [ ] Most relevant experience highlighted first
- [ ] Skills section prioritizes required skills
- [ ] Removed irrelevant experience/skills
- [ ] Achievements relate to job requirements
- [ ] Cover letter written (if required)

---

## Before Sending
- [ ] Correct company name in cover letter
- [ ] Correct job title in cover letter
- [ ] Correct contact person name (if known)
- [ ] CV saved with professional file name
- [ ] Cover letter saved with professional file name
- [ ] Both documents in correct format (.pdf or .docx)
- [ ] Email subject line professional
- [ ] Email body professional and brief
- [ ] All attachments included
- [ ] Sent to correct email address

---

## Post-Application
- [ ] Note application date in spreadsheet
- [ ] Set reminder to follow up (1-2 weeks)
- [ ] Save job description for interview prep
- [ ] Update LinkedIn profile if needed
- [ ] Continue applying to other roles

---

## Quick Reference: Common Mistakes
❌ **AVOID:**
- Using "I" or "my"
- Listing responsibilities instead of achievements
- Generic skills without evidence
- Unprofessional email address
- Irrelevant information
- Lies or exaggerations
- Inconsistent formatting
- Typos and errors
- Too much bold/italics/underline
- Cramming too much content

✅ **DO:**
- Use action verbs
- Quantify everything
- Tailor to each job
- Keep it concise
- Be specific
- Proofread multiple times
- Get feedback
- Update regularly
