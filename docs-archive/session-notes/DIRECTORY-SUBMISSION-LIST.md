## Submission Checklist

For each directory, prepare:
- [ ] Company logo (PNG, 512x512px)
- [ ] Screenshots (1200x800px, showing key features)
- [ ] Short description (50-100 words)
- [ ] Long description (200-300 words)
- [ ] Key features list (5-7 bullet points)
- [ ] Pricing information
- [ ] Website URL
- [ ] Social media links
- [ ] Contact email
- [ ] Category tags/keywords
